## Setup

Quick setup guide for the application.

1. Clone Repo

```bash
git clone <repo_url>
```

2. Install dependencies:

```bash
npm install
```

1. Build the project:

```bash
npm run build
```

4. Run with PM2 (and persist):

```bash
npx pm2 start app.js
npx pm2 save
```
